## 概述
宏观上看，本书第1～5章为基础知识篇，涵盖前端开发、深度学习两大技术栈；第6～12章为进阶提升篇，涵盖线性回归、逻辑回归、XOR问题、预训练模型加载、MNIST手写数字识别、OCR识别、人体姿态检测、目标检测等13个案例，所有案例均部署在个人自建平台上。
